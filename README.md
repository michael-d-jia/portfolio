# Portfolio

Personal portfolio site for Michael Jia. Built with Vite + React + Tailwind CSS v4.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:5173.

## Build

```bash
npm run build
npm run preview
```

## Deploy (Vercel)

1. Push this repo to GitHub.
2. On [vercel.com](https://vercel.com), click **Add New → Project** and import the repo.
3. Vercel auto-detects Vite — keep defaults and click **Deploy**.

## Edit content

Most placeholder copy lives in:

- `src/components/Hero.jsx` — name, tagline, intro
- `src/components/About.jsx` — bio paragraphs and skills list
- `src/components/Projects.jsx` — `projects` array (title, description, tech, github, demo)
- `src/components/Contact.jsx` — email and social links
